# BIGANのつかいかた
Markdown記法に準拠。

## BIGAN

システム本体。二個体での比較と自然言語での評価を取り入れた IGA によって生成した模様からなる背景画像を生成する。

### 使い方

1. エクスプローラーから`index.html`をそのまま開いても動作しないので、以下のいずれかの方法で環境を整える
	- VSCodeのLive Serverプラグインで仮想サーバーを立ち上げる（システムは利用可能になるが、評価実験用のアンケートは送信できない）
	- Webサーバーを用意する（Dockerを用いた仮想環境可。ApacheとPHP 8.3が必要。）
2. 対応するディレクトリに`BIGAN`フォルダ以下一式を配置する
3. ブラウザを開き、`index.html`までのパスをアドレスバーに入力すると動作を開始する
4. 「使い方」を読みながら準備完了を待つ
5. 「使い方」を閉じると、評価画面が現れる
6. テキストエリアに評価や要望を入力し、「決定」をクリックすると次の画像が出現する
7. 満足のいく模様が出力された場合、右クリックでダウンロードできる

### ファイル・フォルダの説明

- `index.html`：本システムの動作ページ
- `Dockerfile`：Docker環境で動かす場合の、PHPのバージョンを指定
- `compose.yml`：Docker環境でWebサーバーの最上位階層（/var/www/html/）にマウントするためのファイル
- `check.php`：評価実験アンケートの入力内容を確認するためのページ
- `complete.php`：評価実験アンケートの入力内容が送信完了すると表示されるページ
- `assets/`：ページ内に使われる画像やデータセット
- `assets/dict/`：kuromoji.jsの辞書ファイルを格納
- `assets/images/`：ページ内に使われる画像ファイルを格納
- `assets/keysFromTopic.csv`：視覚的特徴に関連する語の、当てはまる着眼点を定義するデータセット
- `assets/conditionValues.json`：感覚的評価に関連する語の、感覚の度合いを定義するデータセット
- `assets/colordic.csv`：カラーコードと色名を紐づけるデータセット
- `assets/textReplacementPairs.csv`：同じ意味の表現を統合するデータセット
- `collected/`：評価実験アンケートの送信先
- `collected/form_init.csv`：フォーム入力内容の書き込み先である`form.csv`のバックアップ
- `script/`：本システムの各種処理
- `style/`：本システムのスタイル

## dicEditor

論文3節の「視覚的特徴に関連する語の，当てはまる着眼点を定義するデータセット」および「感覚的評価に関連する語の，感覚の度合いを定義するデータセット」を直感的に編集したいがために作成したツール。データを編集し始めてから10分が経過すると保存を促してくれる。

### 使い方

1. BIGAN-使い方 1～3 と同様、仮想サーバーか何かを用意して（PHPは不要）移植し、ブラウザでページとして開く
2. `topic.html`を開いた場合は`BIGAN/assets/keysFromTopic.csv`を、`condition.html`を開いた場合は`BIGAN/assets/conditionValues.json`をアップロードすると編集モードになる
3. ファイルを保存する際は、画面の右上に「保存」ボタンがあるのでクリックする

### ファイル・フォルダの説明

- `topic.html`：keysFromTopic.csvを編集できる
- `condition.html`：`conditionValues.json`を編集できる
- `script/`：本ツールの各種処理
- `style/`：本ツールのスタイル
